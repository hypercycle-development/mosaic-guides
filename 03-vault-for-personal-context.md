# Vault: Save Personal Context for Better Responses

Vault lets you store trusted context once, then reuse it across conversations without repeating yourself.

## What to store in Vault

- Project goals and priorities.
- Personal writing preferences.
- Team rules, workflows, or checklists.
- Frequently reused reference notes.

## 1) Create your first box

1. Open **Vault**.
2. Create a box with a clear name (for example, “Marketing Plan Q2”).
3. Add a short description so future-you remembers what belongs there.

Use one box per topic. Avoid mixing unrelated content.

## 2) Add entries that are easy to reuse

Good entries are:

- Short and specific.
- Written as facts or decisions.
- Updated when they change.

Avoid long, unstructured text dumps when possible.

## 3) Grant access intentionally

If your app setup supports per-agent access:

1. Give each agent access only to relevant boxes.
2. Keep sensitive boxes limited.
3. Review access when you add new agents.

Least access usually means safer and cleaner outputs.

## 4) Use Vault in real chats

When prompting in AI Chat, refer to Vault context clearly, for example:

- “Use the style guide from my Vault writing box.”
- “Plan next steps based on the project box.”

This gives more consistent results than retyping context each time.

## Vault habits that save time

- Update entries right after decisions are made.
- Archive outdated entries so the assistant doesn’t use stale info.
- Keep naming conventions simple and consistent.
