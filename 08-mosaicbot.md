# Mosaic Bot: Run an Always-On Assistant

Mosaic Bot is for users who want proactive help, not just on-demand prompts.

<img width="1229" height="762" alt="image" src="https://github.com/user-attachments/assets/166cec32-7e9d-4e12-8249-cfe45e44b2ae" />

## Good use cases

- Daily check-ins and reminders.
- Monitoring recurring workflows.
- Keeping long-running tasks moving forward.

## 1) Open Mosaic Bot and pick one goal

1. Open **Mosaic Bot**.
2. Choose a single clear objective for your first setup.
3. Keep scope small for the first run.

Example: “Morning summary of open priorities.”

## 2) Configure routine behavior

Set a cadence that matches your workflow:

- Daily for planning.
- Multiple times per day for fast-moving projects.
- Weekly for reporting or reflection.

Start less frequent, then increase if useful.

## 3) Connect supporting context

For better outputs, combine Mosaic Bot with:

- Vault context (your rules and priorities).
- Useful tools (only what it needs).
- Optional connected sources (like MCP/Gmail where relevant).

The clearer the context, the better the bot’s guidance.

## 4) Review and tune results

- Check if updates are actionable.
- Reduce noise by tightening prompts.
- Refine goals every few days.

Treat the first week as setup and tuning.

## Practical safety tips

- Keep important actions human-reviewed.
- Avoid over-automation on critical workflows.
- Prefer incremental changes over major reconfiguration.
