# Start Here: Your First 10 Minutes

If this is your first time using MosAIc, this guide helps you get from opening the app to sending useful AI prompts with confidence.

## What you’ll do

- Open the app and understand the main areas.
- Set up at least one AI agent.
- Send your first message in AI Chat.
- Learn where the major features live.

## 1) Learn the layout quickly

When MosAIc opens, focus on three areas:

- **Sidebar**: your feature menu (Chat, Vault, Tool Sandbox, and more).
- **Main content area**: where each page appears.
- **Bottom input bar**: fastest way to start a chat prompt.

Tip: Keep the Sidebar open while learning. It makes switching features easier.

## 2) Configure an agent

1. Open **Configuration** from the Sidebar.
2. Add or enable an agent profile.
3. Test the connection if a test button is available.
4. Save changes.

If no agent is active, many AI actions will be limited.

## 3) Send your first AI prompt

1. Open **AI Chat**.
2. Select an active agent.
3. Ask a practical question, for example:
   - “Summarize my priorities for this week.”
   - “Turn these notes into a checklist.”
4. Refine the result with a follow-up prompt.

Use short, direct follow-ups for better results.

## 4) Explore the main features

Try these next, one at a time:

- **Vault**: store personal context for future answers.
- **Tool Sandbox**: install extra tools.
- **Gmail**: connect email workflow.
- **MCP Servers**: bring in external tools/data.
- **Chat Rooms**: collaborate with people + agents.
- **Mosaic Bot**: keep an assistant running in the background.
- **Web3**: manage wallet and transfers.

## 5) First-week habits

- Keep one main agent active for consistency.
- Save reusable info in Vault early.
- Review tool permissions before using new tools.
- Use small test actions before large actions (especially in Web3).
