# AI Chat: Ask Better Questions and Get Better Answers

AI Chat is where most users spend their time in MosAIc. This guide helps you get consistently useful responses.

## What this feature is best for

- Turning messy notes into clear summaries.
- Brainstorming ideas and plans.
- Drafting messages, posts, and documents.
- Running repeatable workflows with connected tools.

## 1) Start a focused chat

1. Open **AI Chat**.
2. Pick an active agent.
3. Give clear context in 1–3 lines.
4. Ask for a specific output format (list, table, short plan, etc.).

Example:

“I’m planning a launch this month. Give me a 2-week checklist with priorities.”

## 2) Use the prompt pattern that works

Use this pattern:

- **Context**: what this is about.
- **Goal**: what you want.
- **Constraints**: tone, length, deadline, format.

Short prompts are fine, as long as they are specific.

## 3) Improve answers with follow-ups

When the result is close but not perfect, ask follow-ups like:

- “Make this shorter.”
- “Turn this into action items.”
- “Rewrite for a non-technical audience.”
- “Give 3 alternatives.”

You usually get better results faster by refining than starting over.

## 4) Keep high-quality chats reusable

- Save strong prompts in a note.
- Reuse the same structure for repeated tasks.
- Move important background info to Vault so you don’t repeat it every time.

## Common mistakes to avoid

- Asking for too much in one message.
- Missing constraints (length, audience, style).
- Switching topic mid-thread without saying so.
