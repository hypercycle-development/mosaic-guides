# Agents & Settings: Make the Assistant Work Your Way

This guide helps you set up MosAIc so responses match your goals, style, and daily workflow.

## What you can control

- Which agents are active.
- Which model/provider each agent uses.
- App behavior and preferences.

## 1) Create a clean agent setup

1. Open **Configuration**.
2. Keep only the agents you actually use.
3. Give each agent a clear purpose name (for example, “Writing,” “Research,” “Ops”).
4. Disable unused agents to reduce confusion.

Clear agent roles improve consistency.

## 2) Test before relying on an agent

After creating or editing an agent:

1. Run a connection test if available.
2. Send a short real prompt in AI Chat.
3. Confirm response quality and speed.

If one agent is unreliable, keep a backup agent active.

## 3) Personalize your app experience

In settings, adjust preferences that affect daily use:

- Theme and visual comfort.
- Navigation behavior.
- Any optional UI controls you prefer on/off.

Make small changes, then use the app for a day before changing more.

## 4) Keep your setup stable

- Avoid frequent model switching unless needed.
- Keep one “default” agent for most tasks.
- Use specialist agents only when their role is clear.

## Quick maintenance checklist

- Remove or disable stale agents.
- Re-test after changing provider credentials.
- Keep naming clear and purpose-driven.
