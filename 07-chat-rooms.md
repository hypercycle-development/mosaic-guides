# Chat Rooms: Collaborate with People and AI Together

Chat Rooms are useful when you want group collaboration instead of one-on-one chat.

<img width="1230" height="762" alt="image" src="https://github.com/user-attachments/assets/63d4fcb6-4b6e-4051-8b32-460d47b48d20" />

## What makes Chat Rooms valuable

- Shared context for teams.
- Faster decision-making in one thread.
- AI support inside group conversations.

## 1) Create or join a room

1. Open **Chat Rooms**.
2. Create a room (or join an existing one).
3. Give the room a clear purpose name.

Examples: “Weekly Planning,” “Support Triage,” “Launch War Room.”

## 2) Set room expectations early

At the top of a new room, post a short “how we use this room” message:

- Purpose of the room.
- Expected response time.
- Decision format (for example, “reply with ✅ or ❌”).

This keeps collaboration organized.

## 3) Use AI in a team-friendly way

Ask AI to do group-friendly tasks:

- Summarize long threads.
- Extract decisions and open questions.
- Turn discussion into next steps with owners.

Request concise outputs so everyone can scan quickly.

## 4) Keep rooms healthy over time

- Close stale topics with a final summary.
- Create new rooms when a topic becomes too broad.
- Archive or stop using rooms that no longer serve a purpose.
