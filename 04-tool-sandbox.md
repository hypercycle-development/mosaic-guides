# Tool Sandbox: Add New Capabilities Safely

Tool Sandbox is where you add extra features to MosAIc, while keeping control over what those tools can do.

## Why this matters

- You can expand what the assistant can do.
- You can review permissions before trusting a tool.
- You can disable or remove tools you no longer need.

## 1) Start with one tool

1. Open **Tool Sandbox**.
2. Install one tool that solves a clear problem you have.
3. Review its details and enable it.

Avoid installing many tools at once until your workflow is stable.

## 2) Review permissions before using

Before activating a tool, check what access it asks for.

- If access seems broader than expected, skip it.
- Prefer tools with focused, understandable permissions.

If a permission request doesn’t match the tool purpose, treat it as a red flag.

## 3) Test with low-risk tasks

In AI Chat, run a small test:

- Ask the tool to perform a harmless sample action.
- Confirm output quality.
- Decide if it belongs in your regular workflow.

## 4) Keep your toolbox clean

- Pin the tools you use often.
- Disable tools you rarely use.
- Remove tools you no longer trust or need.

Fewer, higher-quality tools usually produce better results.

## Common mistakes to avoid

- Installing tools without reading permission prompts.
- Using new tools on important tasks first.
- Letting old tools remain enabled “just in case.”
