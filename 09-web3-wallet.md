# Web3 Wallet: View Assets and Send Transfers Carefully

Web3 in MosAIc helps you manage wallets and transfers with a clear interface, but this area needs extra caution.

## What this feature is for

- Viewing wallet addresses and balances.
- Managing saved addresses.
- Sending transfers carefully with human confirmation.

## 1) Start with read-only checks

1. Open **Web3**.
2. Verify your active wallet and network.
3. Review balances before doing anything else.

Always confirm you’re on the intended network.

## 2) Build a safe address workflow

- Save frequently used addresses with clear labels.
- Double-check each address before first use.
- Keep a trusted list and avoid copy/paste errors.

Label quality reduces transfer mistakes.

## 3) Send small test transfers first

Before large transfers:

1. Send a small test amount.
2. Confirm it arrives correctly.
3. Then send the full amount.

This single habit prevents many costly errors.

## 4) Use AI assistance carefully

AI can help explain steps and format information, but you should still manually verify:

- Recipient address
- Amount
- Network
- Fees

Never treat financial actions as fully automatic.

## High-confidence safety checklist

- Confirm network.
- Confirm address.
- Confirm amount.
- Confirm final transaction details.
