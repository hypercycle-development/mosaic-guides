# MCP Servers: Connect External Tools and Data

MCP Servers let you extend MosAIc with external tools and data sources, so your assistant can help with more than text-only tasks.

## When to use MCP

- You need access to outside systems or data.
- You want one assistant workflow across multiple tools.
- You want reusable, connected automations.

## 1) Add your first server

1. Open **MCP Servers**.
2. Add one server connection.
3. Save and confirm it appears as active.

Start with one reliable server before adding more.

## 2) Validate with a simple task

In AI Chat, run a low-risk test task that uses the server.

- Keep the request specific.
- Confirm the output format is useful.
- Repeat once to check consistency.

If output is unclear, narrow your prompt and try again.

## 3) Build a repeatable prompt

Create a reusable prompt template for MCP tasks:

- Goal
- Input details
- Required format
- Success criteria

This keeps results stable and easier to review.

## 4) Keep control and clarity

- Prefer trusted servers.
- Remove old or unused connections.
- Review results before taking important actions.

Treat MCP as a productivity multiplier, not an autopilot.
